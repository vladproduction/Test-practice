# Spring Pet Clinic 

This source code repository contains JUnit 5 and Spring Framework Testing examples.

The main source code is a copy of the [Spring Framework Pet Clinic project](https://github.com/spring-petclinic/spring-framework-petclinic). 
This is a reference project using a traditional 3-layer architecture without Spring Boot.

## Setup
### Requirements
* Should use Java 11 or higher. Previous versions of Java are un-tested.
* Use Maven 3.6.0 or higher

### Running Locally
After cloning this repo, from the project root run:
```text
./mvnw jetty:run-war
```
